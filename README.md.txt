# CalmSpace 🌸

CalmSpace is a mental health support web application designed to help users track moods, practice mindfulness, write journals, and access calming resources in one peaceful space.

## Features ✨
- Mood Tracker
- Journal Entries
- Mindfulness & Yoga Section
- Motivation & Detox Tips
- Emergency Helplines
- Mood-based Music Suggestions
- Dashboard with Mood Trends
- Interactive UI with Soft Purple Theme

## Tech Stack 💻
- HTML
- CSS
- JavaScript

## Future Improvements 🚀
- Login/Signup Authentication

- AI Chatbot Support
- Personalized Recommendations
